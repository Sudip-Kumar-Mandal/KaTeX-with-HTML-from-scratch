    • LaTeX

  • Software system for document preparation.

  • early 1980s

  • became a standard for creating publications particularly in academia

  • one-stop-shop for publishers, allowing users to create covers, table of contents, indexes, bibliographies and more.

  • massive software package with 100s of files

  • Created for paper publications, before the WWW

    • KaTeX

  • A fast, lightweight JavaScript library for rendering math equations on the web using the TeX typesetting standards.

  • created in 2014 by Emily Eisenberg and Sophie Alpert, while they were working at the Khan Academy

  • K.A presumably refers to the Khan Academy

  • allows users to use the established TeX syntax in plain HTML documents, KaTeX converts the TeX syntax into MathML and render it using CSS to match the TeX typesetting.

    • MathML

  • Mathematical Markup Language

  • used for describing mathematical notations and capturing both its structure and content

  • been around since 1998 and is currently part of HTML5

  • cumbersome to use and doesn't look very good.

  • only way to use math equations using HTML

  • so, there are several libraries that will convert TeX syntax into MathMLa and render it using TeX typesetting

  • KaTeX is one of those libraries
